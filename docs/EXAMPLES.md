# Examples Template

Use this space to collect runnable examples that demonstrate core workflows and validate the system end-to-end.

## Quickstart Examples
Provide the smallest possible example to prove the system works (hello world for the stack).

## Feature Walkthroughs
For each major feature, include:
- Goal of the example
- Prerequisites
- Step-by-step commands or UI actions
- Expected outputs/screens

## Troubleshooting Examples
Show how to deliberately trigger and recover from known failure modes for learning and testing.

---
## Instructions for AI Agents
1. Favor copy/pasteable snippets with expected outputs.
2. Keep examples in sync with code; remove or update stale ones promptly.
3. Link to tests that cover the demonstrated flows when available.
