## Instructions for AI Agents

When designing architecture:
1. Start with a simple, minimal viable architecture
2. Prefer standard patterns and established technologies
3. Don't over-engineer for scale until needed
4. Document key decisions as ADRs
5. Consider testability and maintainability from the start
