# Scripts

Store utility scripts for local development or automation (e.g., linting, data seeding, maintenance). Document invocation and expected environment.

## Available scripts

- `./scripts/lint.sh` — placeholder lint entrypoint to be replaced once a stack is chosen.
- `./scripts/test.sh` — placeholder test entrypoint to wire up future test runners.
- `./scripts/check_docs_no_todo.sh` — scans required docs/templates for lingering `TODO:` markers and fails if found.
